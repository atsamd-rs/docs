Type class for all possible Spi types

This trait uses the AnyKind trait pattern to create a type class for Spi types. See the AnyKind documentation for more details on the pattern.

In addition to the normal, AnyKind associated types. This trait also copies the Sercom, Pads, OpMode, CharSize and Word types, to make it easier to apply bounds to these types at the next level of abstraction.
