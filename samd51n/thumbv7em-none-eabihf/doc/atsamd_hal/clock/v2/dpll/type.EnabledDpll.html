An Enabled Dpll

As described in the clock module documentation, the Enabled wrapper implements compile-time clock tree safety by tracking the number of consumer clocks and restricting access to the underlying Dpll to prevent modification while in use.

As with Enabled, the default value for N is U0; if left unspecified, the counter is assumed to be zero.
