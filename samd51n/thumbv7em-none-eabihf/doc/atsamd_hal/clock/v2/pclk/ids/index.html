Module atsamd_hal::clock::v2::pclk::ids

source · [−]

Module containing only the types implementing PclkId

Because there are so many types that implement PclkId, it is helpful to have them defined in a separate module, so that you can import all of them using a wildcard (*) without importing anything else, i.e.

use atsamd_hal::clock::v2::pclk::ids::*;
